So far, we have successfully completed the core development phases of the project. We began with dataset collection using the UNSW-NB15 dataset, followed by extensive data preprocessing, including handling missing values, encoding categorical features, and feature scaling. We then implemented a baseline Artificial Neural Network (ANN) model to establish initial performance. Building on this, we developed an advanced hybrid CNN-LSTM model capable of capturing both spatial and temporal patterns in network traffic, achieving an accuracy of around 97%. Currently, we are working on extending the model to support multi-class attack classification, performing detailed evaluation using metrics such as precision, recall, and F1-score, and preparing visualizations
