Introduction:
With the rapid expansion of digital technologies and internet-based services, computer networks have become increasingly vulnerable to cyber threats such as malware, denial-of-service attacks, and unauthorized access. Traditional intrusion detection systems rely on predefined rules and struggle to identify new or evolving attack patterns. To overcome these limitations, this project proposes a deep learning-based Network Intrusion Detection System that leverages hybrid CNN and LSTM models to analyze network traffic, learn complex patterns, and accurately distinguish between normal and malicious activities, thereby enhancing overall network security.

Problem Statement:
Modern networks face a rapidly increasing number of sophisticated cyber attacks, while traditional intrusion detection systems based on static rules and signatures are unable to effectively detect unknown or evolving threats and often produce high false positives. This project addresses this challenge by developing a deep learning-based Network Intrusion Detection System using a hybrid CNN-LSTM architecture to accurately analyze network traffic and identify malicious activities, improving detection accuracy and adaptability to modern cybersecurity threats.
